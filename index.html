Online Resume

In this project, I created JSON objects to store resume data and employed JavaScript and jQuery to populate the HTML with this content. The resume sections (bio, work experience, projects, and education) are all modular with a separate JSON object and display function for each. Additionally, a Google Map is displayed with pins to mark locations culled from the JSON data.

Arcade Game

This project is a Frogger-style arcade game designed to run in a web browser window. I employed object-oriented JavaScript to create Player and Enemy objects, and coded all the methods needed to deploy them. The game engine and a set of art images were provided. I expanded the basic game functionality by adding Power-Up Gems, which randomly appear during play. When the Player steps on a gem, the gem changes color for a brief period of time and all Enemies move at half speed. Once the gem disappears, Enemies resume moving at their regular speeds.

Web Optimization

In this project, I improved the performance of a provided mobile portfolio. This project consisted of two optimization tasks. First, I optimized the critical rendering path of the index.html page so that it achieved a PageSpeed Insights score of 95/100 for Mobile and 97/100 for Desktop. Second, I optimized the views/pizza.html page so that it rendered with a consistent frame rate of 60 fps when scrolling. I also I reduced the time to resize the pizzas to less than 5 ms when using the pizza slider. For further details on the modifications made to achieve these results, see my github project page.

Neighborhood Map

In this project, I created a neighborhood map using Google Maps and other APIs. This application employs a MVVM (Model-View-ViewModel) framework implemented with Knockout. Knockout uses observables to monitor changes to the model and automatically updates the places menu with this information. Points of interest are highlighted with markers on the map. An information window is displayed containing additional details about each selected location. In addition to Google Maps, Wikipedia APIs and Foursquare APIs are also employed to generate place details. All APIs are called using jQuery AJAX methods.

Feed Reader Testing

In this project, I was given a web-based application that reads RSS feeds. The application included Jasmine and an incomplete test suite. I used Jasmine to write the remaining specifications to test the application. These check the underlying logic as well as event handling and DOM manipulation. All tests are located in jasmine/spec/feedreader.js. The Jasmine results are displayed below the application, listing the tests performed and indicating which ones met the stated specifications. When a test fails, Jasmine produces an error message and trace for the failure.

Author Page

This project is entirely my own creation, an author webpage. I designed and coded these pages to showcase my creative writing projects. I employed flexbox and media queries to create a responsive webpage that displays on both mobile and laptop. Story "data" is represented as JSON objects. My "Home" page is implemented using a MVVM framework with KnockoutJS to bind the data to the DOM elements. This is a work-in-progress, as I intend to add additional pages and features in the future.
