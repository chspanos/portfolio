Online Resume

In this project, I created JSON objects to store resume data and employed javaScript and jQuery to populate the HTML with this content. The resume sections (bio, work experience, projects, and education) are all modular with a separate JSON object and display function for each. Additionally, a Google Map is displayed with pins to mark locations culled from the JSON data.
