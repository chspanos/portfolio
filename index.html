Online Resume

In this project, I created JSON objects to store resume data and employed JavaScript and jQuery to populate the HTML with this content. The resume sections (bio, work experience, projects, and education) are all modular with a separate JSON object and display function for each. Additionally, a Google Map is displayed with pins to mark locations culled from the JSON data.

Arcade Game

This project is a Frogger-style arcade game designed to run in a web browser window. I employed object-oriented JavaScript to create Player and Enemy objects, and coded all the methods needed to deploy them. The game engine and a set of art images were provided. I expanded the basic game functionality by adding Power-Up Gems, which randomly appear during play. When the Player steps on a gem, the gem changes color for a brief period of time and all Enemies move at half speed. Once the gem disappears, Enemies resume moving at their regular speeds.
